# PaceFrame

Mobile app for sharing a customized image containing Strava activity data.

## Planned Features

- Strava login
- Activity sync (with force sync button)
- Activity list
- Creating new frame
  - Choose activity
  - Choose layout aspect-ratio (insta story, insta post, custom, ...)
  - Choose background (image / color / pattern)
  - Choose / position / customize stats blocks
    - Textual Stats (Watts, duration, Heart-rate, ...)
    - Route outline
  - Share finished image
  
## Tech stack

- Flutter
  - strava_client
  - Canvas rendering
